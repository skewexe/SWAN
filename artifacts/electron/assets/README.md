# Electron Assets

Place the following icon files here before building:

- `icon.ico` — Windows icon (256×256 px minimum, ICO format)
- `icon.png` — Linux icon (512×512 px PNG)
- `icon.icns` — macOS icon bundle

You can use an online converter like https://convertio.co to convert
the Swan GMAO logo PNG to ICO and ICNS formats.

The source logo is at: `attached_assets/ChatGPT Image 30 avr. 2026, 11_42_07.png`
